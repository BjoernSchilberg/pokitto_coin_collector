# Pokitto C++ Template

Checkout:

```shell
git clone --recursive https://github.com/BjoernSchilberg/pokitto_cxx_template
```

Build:

```shell
cd pokitto_cxx_template
make
```

## Test

Test with <https://github.com/felipemanga/PokittoEmu>.

```shell
PokittoEmu BUILD/firmware.bin
```

## Optional

Update submodule.

```shell
cd PokittoLib
git checkout master
git pull origin master
```
